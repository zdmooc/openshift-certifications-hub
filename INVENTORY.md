# Inventaire Complet des Dépôts OpenShift

**Auteur : Zidane Djamal**

Ce document répertorie l'ensemble des **35 dépôts** du portfolio GitHub [zdmooc](https://github.com/zdmooc) identifiés comme pertinents pour la préparation aux certifications Red Hat OpenShift.

---

## Légende des Niveaux de Pertinence

| Niveau | Signification |
|---|---|
| **DIRECTE** | Dépôt spécifiquement conçu pour préparer une ou plusieurs certifications OpenShift |
| **COMPLEMENTAIRE** | Dépôt qui renforce les compétences nécessaires sans cibler directement une certification |
| **INDIRECTE** | Dépôt qui fournit des bases fondamentales utiles pour l'écosystème OpenShift |
