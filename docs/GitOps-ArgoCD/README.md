# Guide : GitOps & Déploiement Continu avec ArgoCD sur OpenShift

**Auteur : Zidane Djamal**

## Pourquoi GitOps est Essentiel pour les Certifications OpenShift

Le GitOps est devenu le standard de facto pour la gestion des déploiements sur OpenShift. Red Hat intègre nativement **OpenShift GitOps** (basé sur Argo CD) dans sa plateforme. La maîtrise de GitOps est indispensable pour les certifications EX280 et EX288, et constitue un pilier central de l'architecture OpenShift moderne.

Ce portfolio contient une richesse exceptionnelle de ressources sur ce sujet, allant du niveau débutant à l'expertise production-grade.

## Parcours Recommandé

1. **Débutant** : Commencez par `ArgoCD-Complete-Master-Course` pour les concepts de base.
2. **Intermédiaire** : Suivez le programme de 8 semaines de `argocd-complete-masterclass`.
3. **Expert** : Plongez dans `argocd-expert-pack` pour les patterns production-grade.
4. **Mise en pratique** : Déployez `gitops-platform` pour avoir votre propre plateforme GitOps complète.
5. **Standardisation** : Adoptez le standard `openshift-gitops-standard-6files` pour vos projets.
