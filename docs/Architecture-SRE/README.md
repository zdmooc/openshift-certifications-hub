# Guide : Architecture OpenShift & Site Reliability Engineering (SRE)

**Auteur : Zidane Djamal**

## Vers le Titre d'Architecte OpenShift

Le titre de **Red Hat Certified OpenShift Architect** représente le plus haut niveau de certification dans l'écosystème OpenShift. Il nécessite la détention de l'EX280 ou de l'EX288, complétée par 5 certifications de spécialisation supplémentaires.

Ce portfolio contient des ressources de niveau architecte qui couvrent la conception de plateformes, la migration, le SRE et l'observabilité.

## Parcours vers l'Architecture

1. **Obtenez l'EX280 ou l'EX288** en suivant les guides dédiés.
2. **Étudiez** `rh-openshift-architect-lab` pour comprendre le parcours complet sur 18-24 mois.
3. **Construisez votre portfolio** avec `openshift-platform-architect-portfolio`.
4. **Maîtrisez les opérations** avec les runbooks SRE et les blueprints réseau/sécurité.
5. **Choisissez vos 5 spécialisations** en fonction de votre parcours et de vos intérêts.
